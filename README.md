# altair-kitchen-sink

> https://joaopalmeiro.github.io/altair-kitchen-sink/

Sample Altair charts to exemplify Altair themes.

## Development

```bash
conda env create -f environment.yml
```

```bash
conda activate altair-kitchen-sink-dev
```

```bash
jupyter lab
```
